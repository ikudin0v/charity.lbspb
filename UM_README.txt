Алгоритм использования API:
1. грузим все категории - формируем фильтр (categories.fetchAll())
    Важно! фильтр может состоять из блоков, имеющих свои названия (поле group), внутри которых логика 'ИЛИ', между блоками 'И'
    пример:

    цвет: 
    - красный
    - жёлтый
    - синий
    форма:
    - круг
    - квадрат
    материал:
    - пластик
    - металл
    - стекло

    цвет маркеров на карте (поле color) может иметь только одна группа (сервер будет это контролировать)!

2. грузим все локации и организации - формируем карту, цвет точки/кластера согласно цвету категории из п.1 (locations.fetchAll() и organizations.fetchAll())
3. делаем возможность фильтрации по категориям на карте (средствами React, т.е. чисто на клиенте)
4. делаем возможность поиска - запрос на сервер (organizations.search(data)), возвращающий только список id подходящих организаций
    Важно! на клиенте нужно сохранять поисковый запрос и ответ сервера в рамках сессии, т.к. они могут повторяться и повторно не запрашивать!
5. клик на точку на карте - модальное окно с информацией об адресе + список организацией согласно фильтру (средствами React, т.е. чисто на клиенте)
6. отдельная страница со списком всех организаций (organizations.fetchAll())
7. отдельная страница со списком всех персон (persons.fetchAll())
8. отдельная страница одной организации (organizations.getById())
9. отдельная страница одной персоны (persons.getById())
